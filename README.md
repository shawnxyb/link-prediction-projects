# 链路预测课程项目

本仓库整理了高级离散数学链路预测实验的代码、实验报告、实验结果和课堂演讲材料。

## 目录

- `代码/`：Facebook LightGCN、图结构特征 MLP、多数据集公平评测等实验代码。
- `实验报告/`：LaTeX 报告源码、图表资源和最新版 PDF。
- `实验结果/`：汇总表格、对比分析和各组实验输出。
- `演讲/`：课程展示 PPT。

## 说明

报告的最后部分已补充完善，重点强化了后续方向、实验局限和总结结论。为保持仓库轻量，未上传 `__pycache__`、LaTeX 编译中间文件、模型权重 `.pt` 和 split 缓存 `.npz`。
